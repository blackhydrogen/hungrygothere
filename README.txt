Hungry Go There
===============

Our product is a mobile application that allows the user to find all the
well-known/good makan (eating) places in Singapore. A key highlight of our
application is that it allows users to plan a journey (i.e. from A to B) and
recommend makan places en-route, or a short detour from the suggested route.

It can be accessed at http://hungrygothere.appspot.com.
